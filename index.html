My name is Oyebamiji Abidemi Samson. I am a self taught frontend web developer who specialize in the use of tools like HTML, CSS, Bootstrap, SCSS, JavaScript and ReactJs to create beautiful and facinnating web pages and web application to meet the need of my clients or hire.

My most prefered development environment is the Microsoft Visual Studio Code because it combines the simplicity of a source code editor with powerful developer tooling, like IntelliSense code completion and debugging. It is also supported by macOS, Linux and Windows. It gives room to customize every feature to your liking and install any number of third party extensions. This is not to say others like Bracket and sublime text are not good enough as I have also worked with them and find them to be quite good too.

Some of my competencies include Creating website layout by using standard HTML/CSS practices, Writing well designed testable efficient code by using the best software development practices,Integrating graphics, audio and video into the website, Integration of data from various back-end services and databases, Maintaining, expanding and scaling the website, Staying plugged into emerging technologies and applying them into operations and activites of the organiation.

There are several web development tool available and being a web developer i have used many of them like GitHub - to deploy a website and work with a team on a particular project. CodeKit - to minify, combine and syntax-check in JavaScript. Javascript - to structure the web applications and even deploy a one-page application. CodePen - to deploy a website and build test cases. Grunt - for automating repetitive tasks.
